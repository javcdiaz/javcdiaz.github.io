## Reflection
### What went well?
Creating the website was easier than it sounded. The html code required to make the list and links to work was easy to make a template of. 
I was then able to insert the specfic details for each tool where they needed to be within the html code. Another step that went well was finding the tools. I was lucky enough to find two main sites that contain more than half the tools.
### What went wrong?
I ran into some issues during my research and testing the utilities. In my research, I found some utilities that had websites but the download link failed. 
There were also utilities that showed up in research but the download including other software that cause pop ups on computers and ads on internet browsers, like Google Chrome, to show up.
Any utility that met the criteria of the exclusion list was removed as a candidate for the webiste. 
Finding broken websites increase in the time it took to complete the research for the best tools.
During the testing, finding utilities that produced unwanted results led to an increase on the time to complete the project.
### What would I do differently?
Some things that I would do different is add more category types with more utilities. I would also expand the full list to include the MAC (Apple) version of some of these utilities. 
If I had more time, I would also add a list for utilities with good reputation but cost money.
