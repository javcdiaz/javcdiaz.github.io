# Javier Caporal
## Project Purpose
Service Desk technicians should have an internal AI bot to assist them in troubleshooting basic network issues.

## Description
Using ChatGPT Businesss account, we will train ChatGPT to respond to questions from technicians and respond base on internal knowledgebase articles so that all technical steps taken by technician were approved. 

## Desired Results
AI will search all internal documents and if search produces 0 results then offer the option for a google search. 

## Exclusions
Do not write articles as AI training is going on.
