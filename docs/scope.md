### Project Name: 
 IT Utility Site

### Project Purpose:
The purpose of the projec is to provided one consolicated location for all the commonly used tools a support represantivie may need. The list is full of free source tools to download. Anyone needing to troubleshoot a computer problem can come to this site and download the desired tool. Whenever you have an issue to troubleshoot on the computer, it can sometimes be difficult to find the right utility tool online needed for the job. This website will bring useful utilities to a single page.

### Feedback Loops:
Need feedback from any IT enthusiast who has had to troublshoot a computer before including teacher and peers. Someone who has worked on a service desk or support before would be perfect.

### Description:
I will create a repository that will contain many of the top 25 most commonly and useful computer executables that are free sources and available for download. The repository will be organized in tool categories (i.e. network tools, disk space tools) to stay organized. Each category will have 3-5 free source or trial tools with good online reputation. 

### People Resources
Peers and professor Cheri Whalen. 

### Desired Results
Website should save time for any one looking for a troubleshooting tool. It should provide mac and windows versions if availablea. Website should also provide links or citations from original source. Website should entice technicians to save it to their favorites. Desired results will yield 25 visits of the website a month.
